# Descripción

Este repositorio sirve para calcular el punto óptimo del siguiente problema de optimización:
$$\min f(x,y) = \frac{\sin(x^2 + y^2)}{10} + \sqrt{\ln(x^2 + y^2 + 1) + 1}$$

Se utilizan los algoritmos de BFGS (Quasi-Newton) y Región de Confianza para encontrar el punto mínimo de la función objetivo y se compara la eficiencia de ambos
